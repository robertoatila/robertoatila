### CORE Project

#### MarkitosSystem — Billing & Access Control System

> Full-stack invoicing system with role-based access control, login audit trail, and session protection. Built as a TCC (graduation project) with production-ready security practices.

**Architecture:** Java 17 + Spring Boot 3.2 · MySQL 8 · Vanilla HTML/CSS/JS · Spring Security · Serving static frontend from backend

**Key Features:**

- RBAC with four roles: `admin`, `owner`, `financial`, `operator`
- Login audit page (admin-only) with timestamped records
- Session protection with automatic redirect to `login.html`
- Logout with confirmation modal and secure session invalidation
- Conflict-free static asset delivery via Spring Boot resource handler

`Java 17` `Spring Boot 3.2` `Spring Security` `MySQL 8` `HTML` `CSS` `JavaScript` *(🔒 Private repository)*
