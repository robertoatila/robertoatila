### 👨‍💻 Sobre Mim:

Sou um desenvolvedor apaixonado por arquitetura de software e soluções escaláveis. Atualmente me formando como **Técnico em Informática para Internet** e **Desenvolvimento de Sistemas**, foco em criar aplicações que conectam o mundo real à nuvem. Meu projeto mais recente, o **EduGestor v3**, reflete minha transição para o desenvolvimento Full-Stack moderno.

- 🎓 Formação: **Técnico em Desenvolvimento de Sistemas** & **Informática para Internet**.
- 🚀 Especialidade atual: Ecossistema **React + PHP API** integrado com serviços Cloud.
- 🛠️ Experiência com: **Supabase**, **SendGrid API**, **OpenAI** e persistência de dados dinâmica.
- 🤝 Disponível para: Projetos desafiadores e colaborações em sistemas Web/Mobile.
